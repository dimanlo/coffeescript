# CoffeeScript Pomodoro Lab

Учебное веб‑приложение на CoffeeScript, которое помогает отработать технику Pomodoro и параллельно разобраться, как устроен фронтенд-проект от структуры до сборки.

## Описание проекта

Pomodoro Lab запускает таймер «10 секунд работы / 5 секунд отдыха», считает завершенные циклы, показывает прогресс и сохраняет историю в LocalStorage. Проект предназначен для студентов, начинающих фронтендеров и наставников, которые хотят объяснить CoffeeScript на понятном примере.

## Технические требования

- Node.js 18+ и npm 9+ (любой LTS подойдет)
- CoffeeScript CLI 2.7 (ставится автоматически через `npm install`)
- Python 3 (для простого статического сервера `npm run serve`, можно заменить на любой другой)
- Современный браузер (Chrome/Edge 118+, Firefox 119+, Safari 17+)
- ОС: Windows 10/11, macOS 13+, Ubuntu 22.04+

## Структура репозитория

- `src/` — исходники на CoffeeScript (`app.coffee` с логикой таймера)
- `dist/` — собранный JavaScript (создается автоматически, не редактируем руками)
- `assets/` — статические файлы: `index.html`, `styles.css`, звуки
- `testing/` — документация по тестированию и обратной связи
  - `test_plan.md` — план тестирования
  - `test_report.md` — отчет о тестировании
  - `feedback_summary.md` — анализ обратной связи
  - `issues.md` — список ошибок и предложений
- `package.json` / `package-lock.json` — зависимости и npm-скрипты
- `README.md` — документация (этот файл)
- `ROADMAP.md` — дорожная карта разработки документации

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/dimanlo/coffeescript
   cd coffeescript
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Соберите проект один раз (создаст `dist/app.js`):
   ```bash
   npm run build
   ```
4. Запустите локальный сервер и откройте `http://localhost:8000/assets/index.html`: как напишите (npm run serve) нажмете на эту сыллку <-
   ```bash
   npm run serve
   ```
5. Хотите править код и видеть результат сразу — включите режим наблюдения:
   ```bash
   npm run watch
   ```

## Примеры использования

- **Запуск помодоро-пары:** нажмите «Старт», дождитесь сигнала и делайте короткий перерыв. Каждый полный рабочий блок автоматически добавляется в счетчик и историю.
- **Изменение длительности:** в `src/app.coffee` можно поменять значения конструктора `PomodoroTimer`:
  ```coffee
  class PomodoroTimer
      constructor: (@workDuration = 25 * 60, @breakDuration = 5 * 60) ->
          # остальные методы без изменений
  ```
- **Отладка и обучение:** включите DevTools → Console. При загрузке увидите подсказку ` Pomodoro Tracker запущен!`, можно следить за состоянием через `window.pomodoroTimer`.

## Рекомендации по объяснению студентам

1. Покажите, что CoffeeScript компилируется в обычный JS (`npm run build`).
2. Разберите класс `PomodoroTimer`: конструктор, методы `start`, `pause`, `reset`.
3. Используйте `npm run watch`, чтобы ученики видели автоматическую перекомпиляцию.
4. Попросите команду добавить свою «фишку» (другой звук, цвет, длительность) — так ребята закрепят материал.
